Notice to Ubuntu 20.04 users
Dear users,

DispalyLink is currently cooperating (https://gitlab.freedesktop.org/xorg/.../-/issues/1028) with XServer community in order to fix this issue that appeared in Ubuntu 20.04, which was due to an update of xserver-xorg-core (2:1.20.8-2ubuntu2) package.

At the moment we are unable to share when this fix will be available in Ubuntu 20.04. To support our users in the mean time we prepared a development version of fixed X.org server 1.20.8 and we kindly ask you to try it and let us know your feedback on how it works.

Please note: this patch is NOT applicable to AMD-only machines, for which we are currently developing a solution. We will update you when it is available.

You can download the package from here: (https://www.displaylink.com/downloads/file?d=310)

In order to install the fixed version, please unpack zip file, enter it and invoke:
sudo dpkg -i xserver-xorg-core_1.20.8-2ubuntu2_amd64.deb

In order to revert to the unmodified xserver package please run:
sudo apt-get install --reinstall xserver-xorg-core

Thank you for your patience and feedback.

Best regards,
DisplayLink Team
